Sistemas Operativos

El sistema Operativo es el programa que hace que usar tu PC sea mas facil, hace años solo eran letras y numeros los cuales eran codigos y actualmente gracias a Apple los sistemas operativos ya tienen imagen y con ayuda del mouse puedes usarlo muy facil.

Windows

Este sistema operativo es el mas popular mundialmente ya que es muy accesible y facil de utilizar, actualmente estan 2 versiones disponibles las cuales si tienen gran diferencia.
Primero tenemos el Windows 8 con una interfas diseñada principalmente para dispositivos touche ya que tiene un modo de usarse complicado con el mouse (si e que no tienes mucha experiencia).



El otro sistema de Microsoft es Windows 10, el cual es una convinación del windows 8 y el windows 7, ya que regresa el boton inicio pero con la interfas del 8, este sistema tiene en configuraciones la opcion de usarse en touche o en PC normal y es mas facil ya que tiene atajos mas faciles de entender ya que es parecido a los windows anteriores al 8.

MAC OS X

Este sistema Operativo no se puede instalar en cualquier PC solo viene de fabrica al adquirir una computadora Apple (las cuales son mas costosas)
Al utilizar este sistema es mas complejo ya que es completamente diferente al mas usado pero sabiendo lo basico es casi lo mismo.

Linux

Uno de los sistemas operativos Gratis mas famosos, pero uno de los mas complicados de usar ya que no todos los programas que se pueden instalar tienen botones para cerrar el programa o minimizarlo, se ocupan comando los cuales son convinaciones con el teclado.
